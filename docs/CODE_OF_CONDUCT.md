# Code of Conduct

Memry is committed to providing a respectful, safe, and constructive community for everyone who participates in this project.

## Our Standards

We expect contributors and maintainers to:

- Be respectful, patient, and constructive.
- Assume good intent and ask clarifying questions before escalating disagreements.
- Welcome feedback and respond to it professionally.
- Focus on what is best for the project and community.
- Show empathy toward other community members.

## Unacceptable Behavior

The following behavior is not acceptable in project spaces, including issues, pull requests, discussions, commits, and community chats:

- Harassment, abuse, threats, or intimidation.
- Discriminatory language or conduct.
- Trolling, personal attacks, or inflammatory comments.
- Sharing private information without permission.
- Sexualized language, imagery, or unwelcome advances.
- Repeated disruption of discussions or reviews.

## Scope

This Code of Conduct applies within all Memry project spaces and when someone is representing the project in public.

## Reporting

If you experience or witness unacceptable behavior:

- For non-sensitive concerns, open an issue or discussion so maintainers can review it.
- For sensitive concerns, contact the project maintainers privately through the repository maintainers' available contact channels.

Please include relevant context, links, screenshots, or other details that help explain what happened.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct. They may remove, edit, or reject comments, commits, code, issues, and other contributions that violate these standards. They may also temporarily or permanently restrict participation when necessary to protect the community.
