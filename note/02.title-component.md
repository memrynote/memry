## INTERACTION BEHAVIORS

### Title Editing
1. Click anywhere on title text → cursor appears, edit mode
2. Type to edit
3. Click outside OR press Enter → save and exit edit mode
4. Press Escape → cancel edit, revert to previous value
5. Empty title → show "Untitled" placeholder

### Title Auto-resize
- Single line by default
- If title is very long: wrap to multiple lines
- Auto-grow height to fit content
- No manual resize handle

### Emoji Selection
1. Click emoji button → open picker
2. Click emoji in picker → select it, close picker
3. Click outside picker → close without change
4. Click "Remove emoji" → remove emoji, close picker
5. Search filters emojis in real-time

### Keyboard Navigation
- Tab from emoji button to title input
- Enter on emoji button → open picker
- Escape → close picker
- Arrow keys in picker → navigate emojis
- Enter in picker → select focused emoji

Implementation Notes
Key Techniques Used

ContentEditable vs Input

Use contenteditable div for auto-growing title
Or use textarea with auto-resize JavaScript
Both work; contenteditable feels more native


Emoji Picker Options

Build custom (full control, more work)
Use library like emoji-mart (faster, feature-rich)
Recommendation: Use library, customize styling


Click Outside Detection

Use useClickOutside hook for picker
Event listener on document, check if target is inside


Auto-resize Textarea

Set height to 'auto', then to scrollHeight
Run on every input change



Why These Choices
