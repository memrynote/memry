## INTERACTION BEHAVIORS

### Send Message
1. Type in input field
2. Press Enter OR click send button → send message
3. Shift + Enter → new line (don't send)
4. Message appears in chat
5. AI starts responding (streaming)
6. Input clears after send

### Quick Action Click
1. Click quick action button
2. Pre-fill input with action text
3. Auto-send (or wait for user to confirm)

### Scroll Behavior
- New messages auto-scroll to bottom
- If user scrolled up, don't auto-scroll
- Show "↓ New messages" button if scrolled up

### Message History
- Persist within session
- Clear when switching notes (or maintain separate history per note)
- Optional: persist to local storage

## PROPS INTERFACE
```typescript
interface AIAgentTabProps {
  noteId: string;
  noteTitle: string;
  noteContent: string;           // For AI context
  messages: ChatMessage[];
  isLoading: boolean;
  isStreaming: boolean;
  onSendMessage: (message: string) => void;
  onStopGeneration: () => void;
  onRegenerateResponse: (messageId: string) => void;
  onCopyMessage: (messageId: string) => void;
  onFeedback: (messageId: string, type: 'up' | 'down') => void;
  onQuickAction: (action: QuickAction) => void;
}

interface ChatMessage {
  id: string;
  role: 'user' | 'assistant';
  content: string;
  timestamp: Date;
  isStreaming?: boolean;
  isError?: boolean;
}

type QuickAction = 'summarize' | 'explain' | 'findRelated' | 'generateIdeas';
```

## OUTPUT

Create an AIAgentTab component that:
1. Displays chat messages in a scrollable area
2. Shows welcome state with quick actions when empty
3. Renders AI and user messages with distinct styles
4. Supports rich text formatting in AI responses
5. Shows typing indicator while AI is processing
6. Implements streaming text display
7. Provides stop generation button during streaming
8. Handles send via Enter key and button click
9. Auto-grows input field for multi-line messages
10. Shows message actions on hover (copy, regenerate)
11. Handles error states gracefully
12. Is fully accessible
13. Auto-scrolls to new messages

Implementation Notes
Key Techniques Used

Streaming Text

Use Server-Sent Events or WebSocket
Append chunks to message content
Update state incrementally


Auto-scroll Logic

javascript   // Only auto-scroll if user is near bottom
   const isNearBottom = scrollHeight - scrollTop - clientHeight < 100;
   if (isNearBottom) {
     scrollToBottom();
   }

Textarea Auto-resize

javascript   textarea.style.height = 'auto';
   textarea.style.height = Math.min(textarea.scrollHeight, 120) + 'px';
```

4. **Markdown Rendering**

   - Or simple regex-based formatting
   - Sanitize HTML for security
```
